# ntc-jettyserver
ntc-jettyserver is a java web server by Jetty

## Maven
```Xml
<dependency>
    <groupId>com.streetcodevn</groupId>
    <artifactId>ntc-jettyserver</artifactId>
    <version>1.0.4</version>
</dependency>
```
